------------- Aula 2 – Para que serve o Python? -------------

 De onde vem o Python?
 O Python foi criado no final dos anos 1980 por Guido van Rossum, na Holanda. 
 A primeira versão oficial surgiu em 1991.

 Por que esse nome?
 O nome Python não vem da cobra. Guido escolheu esse nome por causa do grupo de comédia britânico Monty Python, que ele gostava muito. 
 Por isso, o Python tem exemplos e documentações com um tom mais descontraído.

 Quais grandes empresas usam Python?
 Python é usado por várias empresas grandes no mundo, como:
 - Google
 - Netflix
 - Instagram
 - YouTube
 - Spotify
 - Amazon
 - Microsoft

 Essas empresas usam Python para desenvolvimento de sistemas, automação, análise de dados, inteligência artificial e muito mais.
 *** Linguagem de uso geral: simples, facil e intuitivo.
 *** Multi-plataforma
 *** Codigo aberto
 
 “Batteries included”, em Python, quer dizer: Python já vem “pronto para usar”, com muitas bibliotecas e ferramentas incluídas por padrão.
 Quando você instala o Python, ele já traz várias funcionalidades sem precisar instalar nada extra, como:
 - trabalhar com arquivos
 - datas e horas
 - matemática
 - listas, dicionários
 - acesso a sistema operacional 
 - internet, JSON, CSV, etc.
 
 
 Python é POO (Programação Orientada a Objetos).
 - Python suporta POO, mas não é só POO.
 - Você pode programar em Python de forma:
	- procedural (passo a passo),
	- orientada a objetos (classes e objetos),
	- e até funcional.

 Em Python, tudo é objeto (números, strings, listas, funções etc.), o que facilita usar conceitos de POO como:
	- classes
	- objetos
	- herança
	- encapsulamento
	- polimorfismo

 Python é uma linguagem que suporta Programação Orientada a Objetos (POO), permitindo o uso de classes, objetos e outros conceitos desse paradigma.

------------- Aula 4 – Primeiros comandos em Python3 -------------

 Agora chegou a hora de aprender os comandos básicos do Python e fazer os primeiros programas em Linguagem Python.
 
 Escrevendo uma mensagem na tela (instrução de texto)
 - Delimitadores padrão: aspas simples (' ') ou aspas duplas (" "), usadas para textos.
 - Todo texto deve ficar dentro de aspas.
 - Toda função fica dentro de parênteses ().
   >>>print(''): comando usado para escrever algo na tela.
 - Instrução: print('Hello, World!')

 Escrevendo números
 - Números são usados para efetuar cálculos.
 - Usamos operações matemáticas: +, -, *, / etc.
	print: comando para mostrar o resultado na tela.
 - Instrução: print(7 + 4)
 - Resultado exibido: 11

 Juntando mensagens
 - print('7' + '4')
 - Resultado: 74
 - * (Os valores estão entre aspas, então são textos.)
 
 Mesclando dados
 - print('Olá', 10, 'Novo!')
 - Resultado: Olá 10 Novo!
 
 Variaveis
 - * Toda variável em Python é um objeto.
 
 Representando uma variável com valor fixo
	nome = 'Alessandra'
	idade = 28
	peso = 57.8
	print(nome, 'tem', idade, 'anos e pesa', peso)
	print('nome', idade, peso) ou 
	print(nome, 'tem', idade, 'idade e pesa', peso)
 Resultado: Alessandra tem 28 anos e pesa 57.8
 
 Representando uma variável com valor flexível
 - Função input(): usada para ler dados digitados pelo usuário.
 
 Exemplo:
	nome = input('Qual o seu nome? ')
	idade = input('Qual a sua idade? ')
	peso = input('Qual o seu peso? ')
	print('nome', idade, peso) ou 
	print(nome, 'tem', idade, 'anos de idade e pesa', peso, 'kilos.')
 Resultado (exemplo):
 Alessandra tem 28 anos de idade e pesa 57.8 kilos.

 Observação: Todo arquivo Python possui a extensão .py.

---------- Aula 6 – Tipos Primitivos e Saída de Dados ----------

 Nessa aula, vamos aprender como funcionam os tipos primitivos no Python e as peculiaridades do int() float() bool() e str(). Além disso, veremos como fazer as primeiras operações com a função print() do Python.

 Variavel: valor1 = int(input('Primeiro número: '))
 Leitura:
	- 'Primeiro número: ' → mensagem exibida ao usuário (string).
	- input() → lê a informação digitada pelo usuário (retorna uma string).
	- int() → converte o valor digitado (que veio como string) para número inteiro.
	- valor1 → variável que recebe o valor inteiro convertido.
	
 O input() lê o valor digitado como texto e o int() converte esse valor para número inteiro.

 Tipos primitivos básicos:
 - int (inteiro): 7, -47, 0, 9875
 - float (números reais ou ponto flutuante): 4.0, 0.076, -15.223, 7.0
 - bool (boleano/valores logicos): true / false.
 - str: caracteres ou string (texto):'olá'. '0.7', 'true'.
 
 Metodo .format()
 O .format() é um método usado para inserir valores dentro de um texto (string) de forma organizada.
 Ele serve para formatar mensagens, substituindo {} pelos valores que você informar.
 
 Para que ele é usado?
 - Evita concatenar strings com +
 - Deixa o código mais legível
 - Permite controlar a forma como o texto aparece
 
 Resultado:
 {} → espaço reservado
 .format(nome, idade) → valores que vão substituir os {} na ordem

 Observação importante:
 Hoje em dia, o Python usa muito f-strings, mas o .format() ainda é importante e muito usado em códigos existentes.
 print(f'Meu nome é {nome} e tenho {idade} anos')
 
 nome = 'Alessandra'
 idade = 28
 print('Meu nome é',nome, 'e tenho',idade, 'anos')
 print('Meu nome é {} e tenho {} anos'.format(nome, idade))
 print('Tenho {0} anos e me chamo {1}'.format(28, 'Alessandra'))
 print(f'Meu nome é {nome} e tenho {idade} anos')
 
 
 Conceito importante sobre tipos primitivod
 - input() → sempre retorna str
 - int(input()) → converte o valor digitado para inteiro
 - O tipo primitivo não é automático, você precisa informar (int, float, str, etc.)
 
 # Para saber o tipo primitivo de uma variável

 # O tipo primitivo deve ser especificado
