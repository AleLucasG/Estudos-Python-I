 De onde vem o Python?
 O Python foi criado no final dos anos 1980 por Guido van Rossum, na Holanda.

 Por que esse nome?
 O nome Python não vem da cobra. Guido escolheu esse nome por causa do grupo de comédia britânico Monty Python, que ele gostava muito. 
 Por isso, o Python tem exemplos e documentações com um tom mais descontraído.

 - Microsoft

 Essas empresas usam Python para desenvolvimento de sistemas, automação, análise de dados, inteligência artificial e muito mais.

 - trabalhar com arquivos

 Python é POO (Programação Orientada a Objetos).

	- orientada a objetos (classes e objetos),

	- classes

	- polimorfismo

---------- Aula 6 – Tipos Primitivos e Saída de Dados ----------

 Nessa aula, vamos aprender como funcionam os tipos primitivos no Python e as peculiaridades do int() float() bool() e str(). Além disso, veremos como fazer as primeiras operações com a função print() do Python.

 Variavel: valor1 = int(input('Primeiro número: '))
 Leitura:
	- 'Primeiro número: ' → mensagem exibida ao usuário (string).
	- input() → lê a informação digitada pelo usuário (retorna uma string).
	- int() → converte o valor digitado (que veio como string) para número inteiro.
	- valor1 → variável que recebe o valor inteiro convertido.
	
 O input() lê o valor digitado como texto e o int() converte esse valor para número inteiro.

 Tipos primitivos básicos:
 - int (inteiro): 7, -47, 0, 9875
 - float (números reais ou ponto flutuante): 4.0, 0.076, -15.223, 7.0
 - bool (boleano/valores logicos): true / false.
 - str: caracteres ou string (texto):'olá'. '0.7', 'true'.
 
 Metodo .format()
 O .format() é um método usado para inserir valores dentro de um texto (string) de forma organizada.
 Ele serve para formatar mensagens, substituindo {} pelos valores que você informar.
 
 Para que ele é usado?
 - Evita concatenar strings com +
 - Deixa o código mais legível
 - Permite controlar a forma como o texto aparece
 
 Resultado:
 {} → espaço reservado
 .format(nome, idade) → valores que vão substituir os {} na ordem

 Observação importante:
 Hoje em dia, o Python usa muito f-strings, mas o .format() ainda é importante e muito usado em códigos existentes.

------------- Aula 7 – Operadores Aritméticos -------------

 Nessa aula, vamos aprender quais são os operadores aritméticos do Python e também sua ordem de precedência dentro de expressões matemáticas. 
 Veja como funcionam os operadores de adição, subtração, multiplicação, divisão, exponenciação e quociente na linguagem Python.
 
 Operadores aritmeticos
 + soma
	- 5 + 2 = 7
 - adição
	- 5 - 2 = 3
 * multiplicação
	- 5 * 2 = 10
 / divisão
	- 5 /  = 2.1 (float)
 ** potência
	- 5 ** 2 = 25
 // divisão inteira
	- 5 // 2 = 2 (5 / 2 = 2.1)
 % resto da divisão
	- 5 % 2 = 1 (5 / 2 = 2.1)
	
 Ordem de procedência
 1: ()
 2: **
 3: *, /, //, %
 4: +, -
 
 Ex: 5 + 3 * 2 = 11
	 3 * 5 + 4 ** 2 = 15 + 16 = 31
	 3 * (5 + 4) ** 2 = 3 * 9 ** 2 = 3 * 81 = 243
	 
 Explicação rápida das formatações
 {} → imprime o valor normalmente
 {:20} → reserva 20 espaços para o texto
 {:>20} → alinha à direita
 {:<20} → alinha à esquerda
 {:^20} → centraliza
 {:=^20} → centraliza e preenche os espaços com =	 
 
 end=' '
 O parâmetro end define como o print() termina a linha.
 Por padrão, todo print() termina com uma quebra de linha (\n), ou seja, o próximo print vai para a linha de baixo.	 
 
 Exemplo:
 print(
  'Resultados: soma: {}, subtração: {}, multiplição: {} e divisão: {}'
  .format(soma, sub, mult, div),
  end=' '
 )
 print(
  'potencia: {}, divisao inteira {}, resto da divisão {}'
  .format(potencia, divisaointeira, rest)
 )
 
 O que acontece:
 - end=' ' → não pula para a próxima linha
 - Em vez disso, o print termina com um espaço
 - O próximo print() continua na mesma linha

 Resumo
 O parâmetro end define o que será colocado no final do print(). 
 Usando end=' ', o próximo print() continua na mesma linha.
 
 Quebra de linha = \n

------------- Aula 8 – Utilizando Módulos -------------

 Nessa aula, vamos aprender como utilizar módulos em Python utilizando os comandos import e from/import no Python. 
 Veja como carregar bibliotecas de funções e utilizar vários recursos adicionais nos seus programas utilizando módulos built-in e módulos externos, oferecidos no Pypi.

 Podemos carregar bibliotecas com funções prontas e utilizar recursos adicionais em nossos programas, seja com:
 - módulos built-in (que já vêm com o Python)

 Formas de importação
 📌 Importação geral (importa o módulo inteiro)
 import bebida
 
 Depois usamos assim:
 bebida.cafe()
 👉 Aqui estamos importando tudo do módulo.
 
 📌 Importação específica (importa apenas algo do módulo)
 from doce import pudim
 
 Agora podemos usar direto:
 pudim()
 👉 Aqui estamos importando somente o que precisamos.
 
 
 A biblioteca math é um módulo built-in do Python que traz funções matemáticas prontas.
 
 📌 Para usar

 Ou importar específico: from math import sqrt(square root (raiz quadrada, em inglês))

 🔹 Potência
 math.factorial(5)   # 120
 
 🔹 Valor absoluto
 math.fabs(-10)   # 10.0
 
 🔹 Constantes importantes
 math.pi   # 3.141592...
 math.e    # 2.71828...
 
 🔹 Funções trigonométricas
 math.sin(x)
 math.cos(x)
 math.tan(x)

------------- Aula 9 – Manipulando Texto -------------
 Nessa aula, vamos aprender operações com String no Python. 
 As principais operações que vamos aprender são o Fatiamento de String, Análise com len(), count(), find(), transformações com replace(), upper(), lower(), capitalize(), title(), strip(), junção com join().

 nome = "Alessandra Lucas gomes"
 
 🔹 📌 Fatiamento (Slicing)
 print(nome[0])          # Primeira letra
 print(nome[0:10])       # Do índice 0 até 9
 print(nome[:10])        # Do início até 9
 print(nome[11:17])      # "Lucas"
 print(nome[11:])        # Do 11 até o final
 print(nome[::2])        # Pulando de 2 em 2
 
 🔹 📌 Análise
 ✅ len()
 print(len(nome))        # Quantidade de caracteres
 
 ✅ count()
 print(nome.count('a'))  # Quantas letras "a"
 
 ✅ find()
 print(nome.find('Lucas'))  # Posição onde começa "Lucas"
 print(nome.find('Maria'))  # -1 (não existe)
 
 🔹 📌 Transformações
 ✅ replace()
 print(nome.replace('gomes', 'Silva'))
 
 ✅ upper()
 print(nome.upper())
 
 ✅ lower()
 print(nome.lower())
 
 ✅ capitalize()
 print(nome.capitalize())

 ✅ title()
 print(nome.title())
 
 ✅ strip()
 nome2 = "   Alessandra Lucas gomes   "
 print(nome2.strip())
 
 🔹 📌 Junção com join()
 print('-'.join(nome))
 resutado: A-l-e-s-s-a-n-d-r-a- -L-u-c-a-s- -g-o-m-e-s
 
 🎯 Extra útil (muito usado)
 Separar o nome: print(nome.split())
 Primeiro nome: print(nome.split()[0])
 Último nome: print(nome.split()[-1])

 ⏳ O que é o sleep() no Python?
 O sleep() serve para pausar o programa por alguns segundos.
 Ele fica dentro do módulo time.
 
 import time
 time.sleep(segundos)
 
 Exemplo simples:
 import time

 print("Iniciando...")
 time.sleep(3)
 print("3 segundos depois...")
 
 Pode usar número decimal: time.sleep(1.5)
 
 🎮 Exemplo mais legal (efeito visual)
 import time

 print("Carregando")
 time.sleep(1)
 print(".")
 time.sleep(1)
 print(".")
 time.sleep(1)
 print("Pronto!")

 Criar efeito de animação
